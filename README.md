
# Latent Tactical Intelligence and Chemistry Optimization Framework for Football

ANN-based football analytics project using representation learning and latent embeddings to identify stylistically similar football players.

---

## Features

- Autoencoder-based latent embeddings
- Cosine similarity player matching
- Grey Wolf Optimizer (GWO) for feature selection
- Football statistical analysis
- Data preprocessing and normalization
- Representation learning for football playstyles

---

## Dataset

FBref / Kaggle football player statistics dataset (2025–2026 season).
Fifa26 Player atttribute rating dataset 

---

## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- TensorFlow / Keras
- Matplotlib

---

## Project Pipeline

Football Statistics  
→ Data Preprocessing  
→ GWO Feature Selection  
→ Autoencoder Training  
→ Latent Embedding Generation  
→ Cosine Similarity Analysis  
→ Similar Player Recommendation

---

## Objective

The project aims to learn hidden football playstyle representations using deep learning and optimized feature selection techniques to build a player similarity engine for scouting and tactical analysis.
